### Termux Setup
**How to use auto install**
- Step 1: Download auto-install script and run
  
    ```
    wget <link>
    ```
- Step 2: Run and select your proot distro
  
    ```
    sh proot-auto-install.sh <distro>
    ```
- Example

  ```
  sh proot-auto-install.sh debian
  ```

### Distro List
**Note: Only for auto install**
- Debian `debian`
- Ubuntu `ubuntu`
- Arch `arch-linux`
